# Swoole Dashboard

* 修改 `.env` 将 `VUE_APP_BASE_API` 设置为 `192.168.1.2:9502`
* 安装依赖库： `npm install`
* 启动服务：`npm run serve`
* 打开浏览器：[http://localhost:8080/](http://localhost:8080/)

> 修改 `.vue` 文件和 `.ts` 文件后不需要重启服务，刷新浏览器即可



## 环境
* Version: typescript 4.2.4
* node -v  v10.15.1
* npm -v  7.21.0

## 前端模版
